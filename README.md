# 🚀 AI HUB

AI HUB is a modern AI chatbot website built using:

- HTML
- CSS
- JavaScript
- Three.js
- Ollama AI

Users can chat with multiple AI models like:

- Llama 3
- DeepSeek
- Mistral
- Gemma
- Phi 3
- Code Llama

without API keys 🔥

---

# ✨ Features

✅ Modern UI  
✅ Three.js animation  
✅ Multiple AI models  
✅ Real AI responses  
✅ Ollama integration  
✅ Responsive design  
✅ No API key required

# ⚡ Setup Project

## 1. Install Ollama

Download:

https://ollama.com

---

## 2. Run AI Model

Example:

```bash
ollama run llama3
```

or

```bash
ollama run deepseek-r1
```

---

## 3. Run Website

Open:

```bash
index.html
```

or use VS Code Live Server.
